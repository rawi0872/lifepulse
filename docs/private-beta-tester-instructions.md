# Life Pulse Private Beta Tester Instructions

## What Life Pulse Is

Life Pulse is a personal operating system for daily tasks, habits, goals, reflection, and life areas. It helps you:

- Plan one priority per day
- Complete one visible action
- Reflect briefly at day's end
- Review the week once a few days are logged

It runs in your browser on phone, tablet, and desktop. No native app required.

## Who This Beta Is For

Trusted testers who:
- Want a focused daily loop, not a feature buffet
- Can give specific, honest feedback
- Understand this is early — things may change

## How To Start

1. Open: **https://lifepulse-sand.vercel.app**
2. Create an account
3. Complete onboarding (2–3 minutes)
4. Land on **Today** — your daily command center

## First 10 Minutes (Recommended)

1. **Today** — add one priority ("What will make today count?")
2. **Today** — add one task or habit (Quick Capture or Habits page)
3. **Today** — write one short reflection (evening)
4. **Goals / Projects** — add one if you want to test planning
5. **Weekly Review** — open and read the sections
6. **Insights** — open and note what shows when data is sparse
7. **Settings** — check available modules and starting mode
8. **Devices** — note it is preview only

## What To Test Over 3–7 Days

- Morning: set one priority, complete one action
- Evening: write a short reflection
- Day 4–7: open Weekly Review, save to Journal
- Try Tasks, Habits, Journal, Goals, Projects
- Check Body, Mind, Finance, Passions, Knowledge, Coach
- Test on phone (390x844) and desktop
- Optional: install as PWA / add to home screen

## What Not To Worry About Yet

- No native iOS/Android app
- No watch app
- No wearable/device sync
- No push notifications
- No offline sync
- No AI automation or external processing
- No bank connections
- No medical/mental health/financial/legal advice
- Some modules are early and may change

## How To Report Feedback

Tell us:
- What confused you?
- What felt useful?
- What felt unnecessary?
- Where did you get stuck?
- What did you expect to happen?
- What device/browser/screen size?
- Screenshots or short recordings are very helpful

Email or message the founder directly.

## Privacy & Safety Notes

- **Do not enter highly sensitive data** (passwords, banking details, medical records) during beta
- Use simple, realistic test data first
- **Finance** = manual tracking only; not financial/tax/investment/debt advice
- **Body & Mind** = wellness/productivity context only; not medical/therapy advice
- **Coach** = rule-based transparency; not an AI therapist, doctor, or advisor
- **Devices** = preview only; no automatic imports
- Life Pulse is private/manual — no AI summaries or external processing for review features
